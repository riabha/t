# PDF A3 Format with Section-Based Display ✅

## Date: March 5, 2026

## Summary
Updated PDF export to A3 size with multi-line section display, while keeping the on-screen dashboard with the original 3-line badge format.

## Changes Made

### 1. PDF Size Changed to A3
- **Before**: A4 landscape (297mm × 210mm)
- **After**: A3 landscape (420mm × 297mm)
- **Benefit**: 41% more width, can fit more content comfortably

### 2. PDF Cell Format: Section-Based Display
Each cell now shows subjects grouped by section in 3 lines:

```
A: RPC, MMS, AH
B: WSS, SSS, MH
C: QSE, ERP, SM
```

**Format**:
- Line 1: `A: [subjects for section A]`
- Line 2: `B: [subjects for section B]`
- Line 3: `C: [subjects for section C]`

### 3. Reduced Font Sizes for PDF
- **Header font**: 7pt → 6pt
- **Body font**: 6pt → 5pt
- **Department code**: 7pt → 6pt
- **Cell padding**: 1.5mm → 1mm
- **Min cell height**: 8mm → 10mm (to fit 3 lines)

### 4. Dashboard Display (Unchanged)
The on-screen dashboard keeps the original format:
- Shows up to 3 subject badges
- Blue badges for theory, green for labs
- "+X more" for additional subjects
- Hover tooltip shows complete list

## Technical Implementation

### PDF Page Setup:
```javascript
const doc = new jsPDF({
    orientation: 'landscape',
    unit: 'mm',
    format: 'a3' // Changed from 'a4'
});
```

### Section Grouping Logic:
```javascript
// Group by section and show in 3 lines
const sections = {};
classes.forEach(cls => {
    const sectionLetter = cls.section_name.slice(-1); // Get A, B, C
    if (!sections[sectionLetter]) {
        sections[sectionLetter] = [];
    }
    sections[sectionLetter].push(cls.subject_code);
});

// Build multi-line cell content
const lines = [];
['A', 'B', 'C'].forEach(letter => {
    if (sections[letter]) {
        lines.push(`${letter}: ${sections[letter].join(', ')}`);
    }
});

row.push(lines.join('\n'));
```

### Font Sizes:
```javascript
headStyles: { 
    fontSize: 6,  // Reduced from 7
    cellPadding: 1 // Reduced from 1.5
},
bodyStyles: { 
    fontSize: 5,  // Reduced from 6
    minCellHeight: 10, // Increased from 8 to fit 3 lines
    cellPadding: 1
}
```

### Centered Content for A3:
```javascript
// Cover page centered at 210mm (half of 420mm)
doc.text('QUEST University', 210, 50, { align: 'center' });

// Footer centered at 210mm
doc.text('QUEST University - Timetable Management System', 210, 270, { align: 'center' });
```
