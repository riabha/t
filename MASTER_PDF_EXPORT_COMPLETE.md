## Technical Implementation

### PDF Generation:
```javascript
// Landscape A4 format
const doc = new jsPDF({
    orientation: 'landscape',
    unit: 'mm',
    format: 'a4'
});

// Color-coded cells
didParseCell: function(data) {
    if (cellText.includes('(Lab)')) {
        data.cell.styles.fillColor = [209, 250, 229]; // Green
    } else if (cellText === 'BREAK') {
        data.cell.styles.fillColor = [241, 245, 249]; // Gray
    } else if (cellText !== '—') {
        data.cell.styles.fillColor = [219, 234, 254]; // Blue
    }
}
```

### Data Processing:
- Loops through all departments
- Fetches timetable data from `allTimetableData`
- Builds grid for each section
- Formats instructor names correctly (lab engineer vs teacher)
- Handles breaks and empty slots

### Auto Table:
- Uses `jspdf-autotable` for tables
- Grid theme for timetables
- Striped theme for statistics
- Custom column widths
- Cell padding and alignment
