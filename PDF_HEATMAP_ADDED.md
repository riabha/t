## Technical Details

### Table Generation:
```javascript
// Build header
const headerRow = ['Dept'];
days.forEach(day => {
    for (let slot = 1; slot <= 8; slot++) {
        headerRow.push(`${day.substring(0, 3)}\nS${slot}`);
    }
});
headerRow.push('Total');

// Build data rows
heatmapGrid.forEach(dept => {
    const row = [dept.deptCode];
    dept.slotData.forEach(day => {
        day.forEach(classes => {
            if (classes.length === 0) {
                row.push('—');
            } else if (classes.length === 1) {
                row.push(classes[0].subject_code);
            } else {
                row.push(`${classes[0].subject_code}\n+${classes.length - 1}`);
            }
        });
    });
    row.push(dept.totalClasses.toString());
    heatmapTableData.push(row);
});
```

### Styling:
- **Font Size**: 6px (compact but readable)
- **Cell Padding**: 1.5mm
- **Min Height**: 8mm
- **Column Width**: Auto-calculated to fit
- **Department Column**: 12mm, bold
- **Total Column**: 10mm, bold

if (cellText === '—') {
